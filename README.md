# 🛍️ BuyBuddy – Smart Product Explorer

## 📌 Project Overview

**BuyBuddy** is a web application that allows users to explore and interact with product data in a simple and intuitive way. It uses a public API to fetch real-time product information and displays it dynamically with powerful features like search, filtering, and sorting.

The application is designed to provide a smooth user experience across all devices while demonstrating core JavaScript concepts and modern UI practices.

---

## 🎯 Objective

The goal of this project is to:

* Understand API integration using `fetch`
* Use JavaScript array Higher-Order Functions (HOFs)
* Build an interactive and responsive UI
* Apply clean coding and modular development practices

---

## ⚙️ Features

### 🔍 Search

* Search products by name or keywords

### 🎛️ Filtering

* Filter products by category
* Filter based on price range

### 🔽 Sorting

* Sort products by:

  * Price (Low → High / High → Low)
  * Rating

### ❤️ Favorites

* Mark/unmark products as favorites

### 🌗 Theme Toggle

* Switch between Dark Mode and Light Mode

### 📱 Responsive Design

* Fully responsive for mobile, tablet, and desktop

---

## 🔗 API Used

This project uses a public API to fetch product data.
Examples:

* Fake Store API
* DummyJSON API

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (or Tailwind CSS / Bootstrap)
* JavaScript (ES6+)
* Fetch API
